# Skepsi (MVP)

Interface web moderna e clean para psicólogo com:
- Login
- Cadastro/lista de pacientes
- Área de prontuário por paciente

## Credenciais seed (MVP)
- Email: **admin@psico.com**
- Senha: **admin123**

## Rodar local
1. Instalar dependências:
```bash
cd web-psicologo
npm install
```
2. Iniciar:
```bash
npm run dev
```

3. Acessar no navegador:
- http://localhost:3000
